## 介绍

基于vue+nuxt.js技术实现的静态博客，作为个人项目，可能会存在一些bug，遇到问题请提 

## 使用

克隆本项目，运行如下命令，即可使用

```bash
# 安装
$ npm install

# 启动
$ npm run dev

# 构建
$ npm run build
$ npm run start

# 生成
$ npm run generate
```
